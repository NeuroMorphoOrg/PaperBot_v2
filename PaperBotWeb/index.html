— PaperBot v1.0 RC 3 —

   PaperBot is a configurable, modular, open-source web-based solution to automatically find and efficiently annotate scientific publications based on full-text searches across publisher portals. Without user interactions, PaperBot retrieves and stores article information (full reference, corresponding email contact, and full-text keyword hits) based on pre-set search logic from scientific literature sources including Wiley, ScienceDirect, Springer/Nature/Frontiers, PubMed/PubMedCentral, and GoogleScholar. Once saved, all information becomes web accessible, allowing efficient triage of articles based on their actual relevance for the user and seamless annotation of suitable metadata dimensions.

Pre-Evaluated Publications {{count['Evaluated']}}

  • Evaluate Publications/Pre-Evaluated Publications has 4 possible modes: The user can evaluate publications and decide on their relevance, and file the associated metadata if needed. The Pre-evaluate publications category is meant for training purposes. If an publications is prescreened as relevant, the publication goes to an intermediate collection where it can be later reviewed in detail. These evaluated publications are directly moved to the designed category (Positive, Negative or Inaccessible). The user can also review pre-evaluate publications, update its data and accept or reject the initial pre-evaluation. It is also possible to add publications manually by typing the full reference, or retrieving the information through the associated PMID or DOI.
